# SimpleCiphers
Программа для демонстрации шифров

## Используемые шифры
* [Цезаря](http://questhint.ru/shifr-tsezarya/)
* [Лозунговый](http://questhint.ru/lozungovyj-shifr/)
* [Полибианский](http://questhint.ru/kvadrat-polibiya/)
* [Порты](http://questhint.ru/bigrammnyj-shifr-porty/)
* [Плейфера](http://questhint.ru/shifr-plejfera/)

## Требования
[.NET Framework 4.5](https://www.microsoft.com/ru-ru/download/details.aspx?id=30653)
